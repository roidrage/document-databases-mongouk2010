# CouchDB #

* Made of the web
* Uses HTTP, JSON, JavaScript
* Queries using Map/Reduce
* P2P-like replication

!SLIDE bullets incremental

# Riak #

* HTTP interface
* Simple data model
* Instant scale out, just add more nodes 

!SLIDE bullets incremental

# MongoDB #

* BSON (binary JSON)
* Dynamic queries
* Index support
* Atomic operations on data
